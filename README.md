Overview
This repository contains a retrospective longitudinal healthcare analytics project evaluating clinical and biochemical response trajectories following Everolimus-based escalation therapy in liver transplant recipients with biopsy-confirmed allograft rejection.
Using reproducible Python-based analytical workflows, the project explores longitudinal biomarker evolution, renal function trends, immunosuppressive exposure dynamics, subgroup treatment-response patterns, and timing-related outcome associations across heterogeneous real-world follow-up intervals.
The project was independently developed using retrospective clinical datasets collected during clinical work within hepatology and transplant services at Dr Rela Institute & Medical Centre.

Clinical Background
Liver allograft rejection remains a major cause of graft dysfunction and post-transplant morbidity despite advances in immunosuppressive therapy.
Although Tacrolimus-based immunosuppression remains the cornerstone of rejection management, selected patients demonstrate persistent graft dysfunction, incomplete biochemical response, progressive cholestatic injury, or concerns regarding calcineurin inhibitor-related toxicity.
Everolimus, an mTOR inhibitor, is increasingly utilized as escalation immunosuppressive therapy in selected patients to facilitate immunological stabilization and calcineurin inhibitor minimization strategies.
This project evaluates longitudinal clinical and biochemical response trajectories following Everolimus escalation using real-world retrospective transplant datasets and exploratory healthcare analytics methodologies.

Clinical Questions Explored
	•	Does Everolimus escalation demonstrate early biochemical stabilization?
	•	Do responders and non-responders demonstrate differing longitudinal graft injury trajectories?
	•	Does delayed Everolimus initiation influence bilirubin outcomes?
	•	Can longitudinal healthcare analytics improve interpretation of heterogeneous transplant follow-up data?
	•	How do renal functional trajectories evolve following immunosuppressive optimization?

Project Objectives
	•	Evaluate longitudinal bilirubin trajectories following Everolimus escalation therapy
	•	Assess hepatocellular injury evolution using AST and ALT trends
	•	Analyze cholestatic injury patterns using ALP and GGT trajectories
	•	Evaluate renal functional stability using serial creatinine measurements
	•	Assess synthetic liver function using albumin and INR trajectories
	•	Analyze longitudinal Tacrolimus exposure dynamics
	•	Compare biochemical trajectories between responders and non-responders
	•	Explore associations between Everolimus initiation timing and biochemical outcomes
	•	Demonstrate reproducible healthcare analytics workflows using Python and real-world transplant datasets
